# Requirements

## v1 (Must Have)
- [ ] Mobile-optimized JHA form that works on phones
- [ ] Job info fields: date, location, task description, crew members, supervisor
- [ ] Hazard analysis table: job step → hazard → control measure (add/remove rows)
- [ ] PPE checklist with common items
- [ ] Signature capture for crew acknowledgment
- [ ] Form submission (persist to backend)
- [ ] Works offline (queue submissions when back online)

## v2 (Next)
- [ ] GC admin form builder with drag-and-drop
- [ ] Conditional field logic (show field X if Y is selected)
- [ ] Custom field types beyond primitives
- [ ] PDF export of completed forms
- [ ] Dashboard for reviewing submissions
- [ ] Analytics on completion rates

## Out of Scope
- User authentication (assume handled externally for v1)
- Multi-language support
- Integration with other construction management tools
- Native mobile apps (PWA is sufficient)
