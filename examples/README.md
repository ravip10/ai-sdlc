# Examples

This directory contains complete worked examples of the AI-SDLC framework in action.

## Available Examples

### jha-safety-forms/
A mobile-first digital Job Hazard Analysis (JHA) form for construction field workers.

**What it demonstrates:**
- Complete PROJECT.md, REQUIREMENTS.md, and ROADMAP.md
- JTBD spec with full flow and edge cases
- Domain rules documentation (OSHA compliance)
- Phase planning with implementation decisions
- React component implementation

**Good for learning:**
- How specs reference domain knowledge
- How planning decisions get captured in CONTEXT.md
- How implementation traces back to specs

---

To use an example as a starting point, copy the relevant files to your project root and customize them.
